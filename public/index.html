Karajan Code

Orchestrate multiple AI agents through one automated pipeline. Code, review, test, secure — in a single command.

GitHub npm Docs

The Pipeline

13 roles chained with quality gates. Deterministic guards, real-time monitoring, stall detection. The loop runs until code is approved or the iteration limit is reached.

discover → triage → researcher → architect → planner → coder → refactorer → sonar → reviewer → tester → security → solomon → commiter
♫

Multi-Agent Pipeline

Chain coder, reviewer, tester, security auditor and more. Each role has its own AI agent and instructions. The orchestrator manages the loop.

✓

TDD Enforcement

When source files change, test changes are required. The coder writes tests first — or the pipeline rejects the iteration.

⚙

SonarQube + SonarCloud

Dual static analysis: SonarQube (local Docker, blocking quality gates) and SonarCloud (cloud, advisory). Both run together in the same pipeline.

⯈

MCP Server — 18 Tools

18 tools exposed via MCP, including standalone roles (kj_discover, kj_triage, kj_researcher, kj_architect). Real-time progress notifications.

★

Review Profiles

Standard, strict, relaxed, or paranoid. Configure how tough the reviewer should be. Solomon role resolves deadlocks.

⚠

Deterministic Guards

Output guard blocks destructive operations and credential leaks. Perf guard catches frontend anti-patterns. Intent classifier pre-triages without LLM cost.

⚖

Smart Model Selection

Triage classifies complexity, then the optimal model is auto-selected per role. Trivial tasks use fast models, complex tasks use powerful ones.

⏲

Session Management

Pause and resume sessions. Fail-fast detection stops wasted iterations. Budget tracking monitors tokens and costs. Expired sessions are auto-cleaned.

✎

Skills Mode

8 slash commands (/kj-run, /kj-code, /kj-review, etc.) with guardrails baked in. No MCP needed — works directly in Claude Code.

MCP Server Included

Use kj as a tool inside your AI agent. Send tasks, get real-time progress via kj_status, receive structured results.

After npm install -g karajan-code, the MCP server auto-registers in Claude and Codex. Or add it manually:

{
  "mcpServers": {
    "karajan-mcp": {
      "command": "karajan-mcp"
    }
  }
}

Install Karajan Code, the multi-agent coding orchestrator. Follow these steps in order: 1. Run: npm install -g karajan-code 2. Run: kj init --no-interactive 3. The MCP server "karajan-mcp" is auto-registered in ~/.claude.json during install.
