## Fichero de Configuración

Fichero de configuración: `~/.karajan/kj.config.yml` (o `$KJ_HOME/kj.config.yml`)

Generado por `kj init`. Ver [Referencia de Configuración](/docs/es/reference/configuration/) para todos los campos.

## Variables de Entorno

| Variable | Descripción |
|----------|-------------|
| `KJ_HOME` | Override del directorio de config/sesiones |
| `KJ_SONAR_TOKEN` | Token de autenticación de SonarQube |
