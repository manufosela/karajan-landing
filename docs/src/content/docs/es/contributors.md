## Como Contribuir

1. Fork del [repositorio](https://github.com/manufosela/karajan-code)
2. Crea una rama de feature (`feat/mi-feature`)
3. Escribe codigo + tests (TDD obligatorio)
4. Envia un Pull Request

Todas las contribuciones son bienvenidas: nuevos agentes, roles, guards, bug fixes, documentacion e ideas.
