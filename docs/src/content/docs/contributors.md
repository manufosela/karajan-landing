## How to Contribute

1. Fork the [repository](https://github.com/manufosela/karajan-code)
2. Create a feature branch (`feat/my-feature`)
3. Write code + tests (TDD enforced)
4. Submit a Pull Request

All contributions are welcome: new agents, roles, guards, bug fixes, documentation, and ideas.
